## 5. 黎曼变换 vs 欧氏加法 — III 为什么稳

### 5.1 为什么稳(三件事)

| 维度 | I(Bridge) | II(Negative Geom) | **III(Manifold)** |
|---|---|---|---|
| 已被验证 | DBIM/CDBM 仅在 image | DiffusionNFT 仅 RL setup | **OFT/BOFT/SVDiff 在 SD 上 3 年成熟** |
| 理论工具 | OT/SDE 数值不稳 | negative bound 常 vacuous | **Lie group / Stiefel 流形 30 年文献** |
| 实现复杂度 | 高(forward+backward SDE) | 中(需 negative 采样器) | **低(Cayley/exp 几行代码)** |
| Failure mode | 训不动、loss 爆 | gradient bias、collapse | 收敛慢但单调下降 |
| DMD 兼容性 | 替换 loss 主项 | 替换 loss 主项 | **只改参数化,DMD 一行不动** |

**III 的关键属性**:不是新损失,只是新参数化。DMD 看到的是同一个 ΔW 起到的效果,只是 ΔW 的生成方式从加法变成流形上的等距变换。

### 5.2 本质区别

#### (1) 几何上:加法在切空间,变换在流形上

- **欧氏加法(LoRA)**:$W' = W + \Delta W = W + BA$,$\Delta W$ 是切空间向量,只在 $\|\Delta W\| \to 0$ 时才正确。
- **黎曼变换(OFT)**:$W' = R \cdot W$,$R \in SO(n)$,任意 $R$ 都把 $W$ 映到流形另一点。

**几何图像**:把 base model 想成地球上一点。LoRA 是给你一根直尺沿切平面戳出去——戳得越远离地球越远。OFT 是给你一个旋转操作,无论旋多少,你还在地球表面上。

#### (2) 不变量上:LoRA 改 spectrum,OFT 保 spectrum

任何线性层的"行为"由两件事决定:**方向**(singular vectors $U, V$)+ **大小**(singular values $\sigma$)。

- **LoRA** $W + BA$:同时改 $U$、$V$、$\sigma$,无约束。
- **OFT** $RW$:把 $U$ 旋到 $RU$,$\sigma$ 完全不变。
- **SVDiff**:反过来,只动 $\sigma$,$U, V$ 不动。

**为什么 spectrum 重要**:网络的 Lipschitz 常数、信号传播稳定性、hyperspherical energy(基础语义结构)都由 spectrum 决定。LoRA 一次更新可能让 $\sigma_i$ 从 1.0 漂到 5.0 → 某方向被异常放大 → catastrophic forgetting。OFT 数学上保证不漂。

#### (3) 度量上:Fisher metric vs Euclidean metric

参数空间真实距离不是 $\|\theta_1 - \theta_2\|_2$,是 **KL 散度** $D_{KL}(p_{\theta_1} \| p_{\theta_2})$。Fisher information $F(\theta)$ 是其二阶近似:
$$
D_{KL}(p_\theta \| p_{\theta + d\theta}) \approx \tfrac{1}{2} d\theta^\top F(\theta) d\theta
$$

- **LoRA SGD**:用 Euclidean metric,每步等间距 $\|\Delta\theta\|_2 = \eta$ → $F$ 大方向迈太大、$F$ 小方向迈太小 → **统计效率低**。
- **Riemannian SGD / Natural gradient**:每步等间距 $\sqrt{\Delta\theta^\top F \Delta\theta} = \eta$ → KL 距离恒定 → **information-theoretically optimal**。
- OFT 因保 Lipschitz 与 spectrum,等价于受限子流形上的廉价 natural gradient 近似。

#### (4) 收敛证明上:Cramér–Rao bound

Few-step finetune 的统计极限是 Cramér–Rao bound,无偏估计方差下界 $F^{-1}$。**只有 Riemannian / natural gradient 下,SGD 渐近达 CRB**;Euclidean SGD 只在 $F = I$ 时才到。

**结论**:**LoRA 数学上不可能是 sample-efficient 最优**,因为根本不在正确的 metric 上做下降。OFT 至少把 metric 拉对了 → "可证明的 sample efficiency"故事成立。

### 5.3 对 motion finetune 的具体含义

- **LoRA 改 motion**:可能在过程中改变 base 视频质量(spectrum 漂 → forgetting)。
- **OFT 改 motion**:旋转 attention representation,不放大不缩小。base 质量理论上保住,只在 motion 维度做"重新对齐"。
- **SVDiff 改 motion**:不旋转方向只缩放,可能不够表达 motion 方向变化。
- **Motion Inversion 的 Q-K vs V 双通道**:暗合此原理 — Q-K 改方向(rotation-like),V 改幅度(scaling-like)— 已经在隐式做 OFT+SVDiff 组合。

### 5.4 论文化命题

> **"Motion finetune on temporal attention should be parameterized as `R Q W_Q` not `W_Q + ΔW_Q`, because motion is a directional re-alignment of attention heads, not a magnitude shift."**

可做 ablation 验证,理论可从 hyperspherical energy preservation 推 bound。
